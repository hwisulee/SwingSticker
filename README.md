# SwingSticker
>자바 Swing, 이미지를 불러와 지정해둔 스티커를 골라 붙여 저장하는 간단한 이미지 스티커 부착 프로그램

<pre>
<code>

설명 : 자바 Swing, 이미지를 불러와 지정해둔 스티커를 골라 붙여 저장하는 간단한 이미지 스티커 부착 프로그램</br>

개발 기간 : '20.06.05 ~ '20.06.15</br>

</code>
